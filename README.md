# Algorithm-Study
알고리즘 스터디입니다.
## 📚 스터디 규칙


1. 레포지토리를 fork한다.
2. 해당 주차의 폴더 안에 자신의 폴더를 만든다.
3. 1주당 알고리즘 3개를 해결하며 각 알고리즘 해결시 Commit 규칙에 맞춰 commit을 한다.
4. 원본 저장소로 PR을 한다.

## 📚 Commit 규칙


- 이름 [해당 주차의 몇번째 알고리즘] [플랫폼] 문제이름
- 예시: Gyulguma [1] [BOJ] 소용돌이 예쁘게 출력하기
- 플랫폼
    - [BOJ] : 백준
    - [PGS] : 프로그래머스
